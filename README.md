# Software Factory Foundation — RFCs

The canonical home for Software Factory Foundation specifications and the RFC drafts that produce them.

Every published standard begins as a public RFC in this repository. Discussion, review, and ratification happen through GitHub Pull Requests and Issues so the full audit trail is preserved.

## Where things live

| Path | Purpose |
| --- | --- |
| [`rfcs/`](./rfcs) | Individual RFC documents, one markdown file per proposal. |
| [`rfcs/TEMPLATE.md`](./rfcs/TEMPLATE.md) | The starting point for a new RFC. |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | The full standards process: stages, voting rules, ratification criteria. |
| [Foundation site](https://software-factory-foundation-site.vercel.app/rfcs) | Browsable index of all RFCs, updated automatically from this repo. |

## Filing an RFC

1. Read [`CONTRIBUTING.md`](./CONTRIBUTING.md) for the full process.
2. Copy [`rfcs/TEMPLATE.md`](./rfcs/TEMPLATE.md) to `rfcs/SFF-RFC-XXXX.md` (use the next available number).
3. Fill in the front-matter and the body sections.
4. Open a Pull Request against `main`. The relevant Working Group will pick it up.
